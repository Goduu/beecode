  type: Debugging
  description: "Identify the error in the following code snippet where the attempt counter is incorrectly updated, and suggest a correction."
  codeSnippet: |
    let numeroSecreto = 5;
    let chute;
    let tentativas = 1;  // Start with 1 since the first attempt is made immediately

    while (chute != numeroSecreto) {
        chute = prompt('Escolha um número entre 1 e 100');
        // Incorrectly placed attempt increment
        tentativas++;
        if (chute == numeroSecreto) { 
            alert(`Isso ai! Você descobriu o número secreto ${numeroSecreto} com ${tentativas} tentativas`);
        } else {
            if (chute > numeroSecreto) {
                alert(`O número secreto é menor que ${chute}`);
            } else {
                alert(`O número secreto é maior que ${chute}`);
            }
        }
    }
  expectedOutput: "The attempt count increments before the user makes a guess, leading to an incorrect count of attempts."
  correctAnswer: |
    Issue: The attempt counter `tentativas++` is incremented before the user makes a guess, which leads to an incorrect count of attempts.
    Correction: Move `tentativas++` to after the `if-else` block to ensure it only increments when the guess is incorrect.
